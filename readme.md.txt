TesteShop - Loja Virtual
Descrição
TesteShop é um projeto de exemplo de uma loja virtual, criado com o objetivo de demonstrar conhecimentos em desenvolvimento web utilizando tecnologias modernas como HTML, CSS, Node.js, Express, SQLite3 e bcryptjs. Este projeto abrange funcionalidades básicas de uma aplicação de e-commerce, incluindo cadastro e login de usuários, persistência de sessão, adição de produtos ao carrinho e visualização de produtos.

Funcionalidades
Página Inicial: Exibe informações gerais sobre a loja e produtos em destaque.
Sistema de Autenticação: Permite que os usuários criem uma conta e façam login com senhas criptografadas.
Carrinho de Compras: Os usuários podem adicionar produtos ao carrinho e visualizar o conteúdo do carrinho em um modal.
Persistência de Sessão: Os usuários permanecem logados durante a sessão até que façam logout.
Conexão com Banco de Dados SQLite3: Gerenciamento de usuários e produtos no banco de dados.

Tecnologias Utilizadas
HTML5: Estruturação do conteúdo da página.
CSS3: Estilização da interface do usuário.
JavaScript (Node.js): Backend da aplicação.
Express.js: Framework web minimalista para Node.js.
SQLite3: Banco de dados leve e local para armazenamento de informações de usuários e produtos.
bcryptjs: Biblioteca para criptografia de senhas.
Instalação

Pré-requisitos
Node.js e npm instalados no seu ambiente de desenvolvimento.
Um editor de código, como VSCode.
